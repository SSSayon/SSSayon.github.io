7月18日

上午有课（致理定制 基础学科交叉实践课程 - 深度学习框架与图形学编程实践），讲了图形学基础。的确讲的是很基础的内容，但也了解了很多。已经感觉这门课不是很吃得消了，主要是要在课后大量地自学相应内容（包括应当有的但我显然没有的前置知识(?)），不然完全没法学。感觉这对我挑战好大。现在就是倾向于当拓展眼界这样子在学了，听说交叉课好像都会自带 P/F (?)，反正也无所谓，夏季学期的退课阶段这门课都上完了，到时候也可以决定记不记 W。

下午宿舍楼断电检修，太热了也没心情，半梦半醒地睡了个下午觉。睡醒后开始看 ODE ，看了存在性定理那一节，解对参数和初始值的连续性。六点左右就吃饭去了。

晚饭吃好又开始照例的摆烂，这破习惯得改掉了！刷到 ChiliChill 北京场的录像，差不多全程看完了。怎么 encore 是山遥路远ww 好羡慕哦。然后开始无意义刷视频。得改了！

七点半/八点左右终于开始学习了，看 Algebra，对称那一章，平面的等距。感觉应该并不难啊，几何意义也挺清楚，但就是不知道为什么看的这么慢。看样子基础不牢地动山摇了x

看到十点多的样子，然后又开始磨洋工磨到快十一点。开始写贵系暑培 Python 的作业。挺简单，学了几个库。效率奇低的到了一点多才写好基本要求和一个 tikz 的 GUI 界面。明天打算再学推荐的 streamlit 第三方库写 GUI。

睡了，明天早上看样子又没了。

7月19日

果不其然，十点的闹铃，将近十一点才从从床上爬下来。迷迷糊糊刚醒的时候玩手机的习惯也得改了，要清醒也不是这样清的！给昨天的 Python 作业改了改说明文档，就去吃中饭了。

下午一点左右开始写 streamlit 的 GUI。界面和流程很快就出来了，然后就发现问题，按下按钮进入新页面，在新页面的输入框输入任何东西后都会直接返回初始界面。一开始认为是输入框在主函数生成还是调用函数中生成的问题（相当于局部变量(?)），搞了好久解决不了。后来终于发现这个库为了实现实时交互，一个特性是检测到任何输入都会重新运行整个程序。所以这就有两方面的问题：第一，如果之前通过点击按钮等手段进入新页面，重置后又不会“模仿”你点按钮的行为而保持这个页面，所以就卡在初始界面了；第二，如果一些输入框/滑动条设置了随机初始值，那么重置后又是一个随机，结果导致你在填完第二个输入框后发现第一个框中的内容随机改了。。。最后依次通过这个库提供的 st.session_state （相当于记录状态的“全局变量”），记录点击事件而保证第二次正常进入新界面；以及在 main.py 中就随机好初始值再传入这个会重新运行的 gui.py ，保证不出现二次随机解决了问题。下午心态是炸的，原本给这种事情并没分配多少时间，毕竟昨天基本都解决了。

结果在交作业的时候又出了问题。作业要 push 到 github 上，因为用了个小号，死活 push 不上去。然后想到本地 git 连的是大号的 github，一看 commit 记录果然都是大号的。。。然后小号的仓库拉了大号做 collaborator 就成功 push 上去了，contributor 就自然而然全是大号，小号一点都没。。。这可不行啊，别还以为我作业是叫别人做的呢x 然后就去学一台电脑怎么配置多个 git 账号，其实倒也和一个账号是一模一样的。这还真是 git 基础不牢地动山摇了（看我昨天说的==）。但至少查漏补缺了一块内容。

搞得我晚饭都没吃。也做了相应的笔记，弄到七点多结束了。这大大超出预算了哈！看来今天数学是没得学咯。然后去看了点 LSGAN 的 Jittor 教程，包括转出去又学了遍几种损失函数。到八点半去了桃李吃夜宵晚饭。

回来后开始摆烂了。。。（喂！）看直播看到快十一点。然后开始跑 MNIST 的 LSGAN。参数都没调，直接 CtrlCV 工程师，20 轮 epoch ，生成出来半像不像数字的东西，但至少能出来是吧。附一张答辩做欣赏。

整体今天比昨天颓，坏！睡了，明天有课。
